## Name

TrainerDex is comprised of two parts: Trainer and Dex. There should never be a space between the two and the D in Dex should be capitalized. 
_The Dex_ is a nickname for the database itself. 

#### Valid
- [x] TrainerDex
- [x] TDX

#### Invalid
- [ ] Trainerdex
- [ ] trainerdex
- [ ] Tdx
- [ ] TDx
- [ ] tdx
